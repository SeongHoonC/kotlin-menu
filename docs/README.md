점심 메뉴 추천 서비스

코치들은 월, 화, 수, 목, 금요일

월, 화, 수, 목, 금 반복
1. 카테고리 무작위 선정
2. 각 코치들은 먹을 메뉴 추천

입력 주의사항 
- 코치 이름
    - 2 ~ 4 글자
- 코치 숫자
  - 2 ~ 5 명

- 각 코치가 못먹는 메뉴
  - 0 ~ 2 개
  - ',' 로 구분함
  - 0개는 빈 값 입력
  - 추천을 못하는 경우는 발생하지 않는다. 
  
- 한 주에 같은 카테고리는 최대 2회
  - 각 코치에게 한 주에 중복되지 않는 메뉴를 추천해야 한다.
  예시)
